This repository contains my solutions to all problems from the **CP-31 Sheet (1100 rating level)** curated by TLE Eliminators. It serves as a comprehensive resource for intermediate competitive programming, building upon fundamental concepts with more complex problem-solving techniques.

- 📊 **Difficulty Level**: 1100 Rating

### Core Concepts
- ✅ **Greedy Algorithms** - Optimal decision making at each step
- ✅ **Dynamic Programming** - Optimization with memoization and tabulation
- ✅ **Two Pointers** - Efficient array traversal techniques
- ✅ **Binary Search** - Fast search and optimization in sorted data
- ✅ **Sorting & Comparators** - Custom sorting strategies
- ✅ **Prefix Sums** - Range query optimization
- ✅ **Constructive Algorithms** - Building solutions step-by-step

### Advanced Concepts
- ✅ **Number Theory** - GCD, LCM, Prime Factorization, Divisors
- ✅ **Bitwise Operations** - Bit manipulation and optimization
- ✅ **String Algorithms** - Pattern matching, palindromes, manipulation
- ✅ **Array Manipulation** - Subarray problems, segment handling
- ✅ **Graph & Game Theory** - Traversal, optimization, game state analysis
- ✅ **Combinatorics & Counting** - Permutations, combinations, counting principles
- ✅ **Data Structures** - Sets, Maps, Stacks, Queues for efficient operations
